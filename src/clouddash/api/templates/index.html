How it works

  1. Triage classifies your intent + extracts entities
  2. Routes to a specialist agent (Technical / Billing / Knowledge / Escalation)
  3. Specialist retrieves KB chunks (hybrid BM25 + dense + LLM rerank)
  4. Cites inline as [KB-XXX § N]
  5. Cross-agent handover preserves context + audit log
  6. Two-layer guardrails: input + output grounding
